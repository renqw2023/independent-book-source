## 🚀 使用方法

### 方法一：直接导入（推荐）
1. 打开legado阅读APP
2. 进入"我的" → "书源管理"
3. 点击右上角"+"号
4. 选择"网络导入"
5. 复制下方对应的书源链接
6. 粘贴到输入框并点击"确定"

### 方法二：下载后导入
1. 下载对应的JSON文件到手机
2. 在legado中选择"本地导入"
3. 选择下载的JSON文件

## ⚠️ 注意事项

1. **网络要求**: 导入时需要稳定的网络连接
2. **存储空间**: 全量合集较大，请确保设备有足够存储空间
3. **性能影响**: 书源过多可能影响搜索速度，建议根据需要选择合适的合集
4. **更新频率**: 书源可能会失效，建议定期更新

## 🔧 技术说明

### 去重算法
- 基于 `bookSourceUrl` 字段进行去重
- 保留第一个出现的书源，移除重复项
- 对于没有URL的书源，全部保留

### 文件格式
- 标准的legado书源JSON格式
- 兼容legado 3.0+版本
- UTF-8编码，确保中文显示正常
