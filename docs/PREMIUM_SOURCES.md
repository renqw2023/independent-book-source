## 🚀 使用指南

### 快速开始
1. 选择需要的书源类型
2. 复制对应的导入链接
3. 在legado阅读APP中选择"网络导入"
4. 粘贴链接并导入

### 特殊配置

#### 起点中文网
1. 导入书源后
2. 使用legado内置浏览器访问 https://www.qidian.com
3. 随便浏览获取Cookie
4. 返回即可正常搜索

#### 晋江文学城
1. 导入书源后
2. 访问 https://m.jjwxc.net/ 并登录
3. 即可使用收藏、VIP购买等功能
