📖 使用方法

  1. 打开legado阅读APP
  2. 进入"我的" → "书源管理"
  3. 点击右上角"+"号
  4. 选择"网络导入"
  5. 复制下方书源链接并粘贴
  6. 点击"确定"完成导入
